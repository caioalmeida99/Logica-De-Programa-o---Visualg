# Exercícios de Lógica de Programação - Visualg

Este repositório contém exercícios desenvolvidos durante meus estudos de lógica de programação utilizando Visualg.

## 📚 Conteúdos praticados
- Estruturas condicionais (SE, SENÃO)
- Estruturas de repetição (PARA, ENQUANTO)
- Variáveis e operadores
- Entrada e saída de dados

## 🎯 Objetivo
Desenvolver raciocínio lógico e fundamentos para programação.
